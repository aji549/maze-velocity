# Maze Velocity

A high-energy Android maze game featuring unlockable vehicles, multiple game modes, and achievements.

## Features

- **Unlockable vehicles and characters**
- **Three game modes:** Maze, Chase (enemy pursues you), Pursuit (you chase enemy)
- **Achievements and progress-based unlock system**
- **Selectable avatars/vehicles, easy expansion**

## Getting Started

1. Clone this repository:
   ```bash
   git clone https://github.com/aji549/maze-velocity.git
   ```
2. Open in Android Studio.
3. Add your vehicle and character PNGs (from the SVGs below) to `app/src/main/res/drawable/`.
4. Build & run on emulator or Android device.

## Extending

- Add maze generation, collectibles, polish visuals/sounds.
- Expand vehicle and character lists.
- Integrate with Google Play for achievements/leaderboards.
